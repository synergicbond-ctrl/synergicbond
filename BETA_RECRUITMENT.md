# BETA_RECRUITMENT

## Goal

Recruit 5-10 real chemistry students and collect production evidence:

- 100+ solves
- 25+ AI solves
- 50+ searches
- 10+ feedback submissions
- 5+ returning users

## Onboarding Instructions

1. Open `https://synergicbond.vercel.app`.
2. Create an account or sign in.
3. Use `/snap-solve` for at least 3 chemistry problems.
4. Use `/search` for at least 5 chemistry topics.
5. Visit `/learn`, `/pyq`, and `/periodic-table`.
6. Submit the beta feedback prompt when it appears.
7. Return the next day and use one feature again.
